# 🎓 HEP Academy Chatbot

Chatbot IA pour étudiants – Mode asynchrone Moodle
AI Chatbot for students – Moodle asynchronous mode

## Fichiers / Files

| Fichier | Description |
|---------|-------------|
| `hep-chatbot.html` | **Fichier principal** – Chatbot complet, standalone |
| `api_proxy.php` | Proxy PHP sécurisé pour cacher la clé API |
| `GUIDE_INTEGRATION_MOODLE.md` | Guide complet d'intégration Moodle (FR + EN) |

## Démarrage rapide / Quick Start

1. Ouvrez `hep-chatbot.html` dans un navigateur pour tester
2. Remplacez `YOUR_ANTHROPIC_API_KEY` par votre clé
3. Suivez le `GUIDE_INTEGRATION_MOODLE.md` pour l'intégration
